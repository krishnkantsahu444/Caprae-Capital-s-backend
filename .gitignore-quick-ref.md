## Quick Commands

```bash
# Check what's ignored
git status --ignored

# Check specific file
git check-ignore -v .env

# See all tracked files
git ls-files

# Remove accidentally tracked file
git rm --cached <file>

# Force add an ignored file (if needed)
git add -f <file>
```
